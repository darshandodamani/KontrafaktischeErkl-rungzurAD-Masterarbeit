A counterfactual explanation answers the question: "What needs to change in an input to get a different AI prediction?"

Example: If an AI system classifies a traffic image as STOP, a counterfactual explanation shows how the image needs to change to be classified as GO.

  • Actionability: Can this change be acted upon?
  • Trust in AI: Does this explanation increase confidence?
  • Visual Coherence: Are only relevant parts changed?
